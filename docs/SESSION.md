# 会话草案

目标：记录一次使用流程，用于后续加入“学习记录导出”和“复盘总结”功能。

## 触发方式

- 用户在任一技能页面完成一次请求。
- 系统记录时间、技能、摘要。
- 用户可在工作区看到最近记录。

## 数据结构（草案）

```json
{
  "timestamp": "2026-03-10T15:10:30",
  "task": "study",
  "summary": "如何在 3 小时内速通高等数学中的极限题型？"
}
```

## 未来扩展

- 增加 `tags`：课程、章节、考试类型
- 增加 `source`：手动输入 / OCR
- 增加 `quality`：自评或系统评分
- 增加 `export`：Markdown 与 CSV
