# UI Rules

## Product Tone

- calm
- concise
- restrained
- useful before expressive

## Visual Rules

- Keep a mild off-white background rather than flat white.
- Avoid noisy gradients and excessive motion.
- Keep forms obvious and readable.
- Preserve the splash line:
  - `人生小满胜万全`
  - `此际我独行`

## Interaction Rules

- First-run setup must be understandable to non-technical students.
- API setup belongs before the main workspace.
- Teaching pages must not use emoticons.
- Companion outputs may use one light emoticon at most.
